كابوينغ للذكاء الاصطناعيشفت الملفات اللي عندك 👌  
عندك الحين `index.html` و `quran.json`، لكن ملف التفسير `tafsir.json` لسه ما انرفع.

ارفعه كذا:
1. وأنت في نفس الصفحة هذي، اضغط زر *Add file* فوق على اليمين
2. اختار *Create new file*
3. في خانة الاسم اكتب: `tafsir.json`
4. الصق هذا الكود فيه:
[
  {
    "surah": 17,
    "ayah": 78,
    "tafsir": "أقم الصلاة -أيها النبي- من وقت زوال الشمس عند الظهيرة إلى وقت ظلمة الليل، ويدخل في هذا صلاة الظهر والعصر والمغرب والعشاء، وأقم صلاة الفجر، وأطل القراءة فيها؛ إن صلاة الفجر تحضرها ملائكة الليل وملائكة النهار."
  },
  {
    "surah": 17,
    "ayah": 79,
    "tafsir": "ومن الليل فتهجد به نافلة لك؛ عسى أن يبعثك ربك مقامًا محمودًا يحمده فيه الأولون والآخرون."
  },
  {
    "surah": 17,
    "ayah": 80,
    "tafsir": "وقل: ربِّ أدخلني فيما هو خير لي مدخل صدق، وأخرجني مما هو شر لي مخرج صدق، واجعل لي مِن لدنك حجة ونصرًا على أعدائي."
  }
]
5. انزل تحت اكتب أي رسالة مثل `add tafsir file` واضغط *Commit new file*

بعد ما يرفع، افتح `tafsir.json` واضغط *Raw* وانسخ الرابط. 
الرابط بيكون:
https://raw.githubusercontent.com/yosfymohmedabdu-ship-it/DigitalQuran/main/tafsir.json
جرب تفتحه في المتصفح. لو طلع لك النص اللي فوق معناته خلصنا ونقدر نربطه بـ FlutterFlow.
