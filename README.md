
# Digital Quran 📖

تطبيق قرآن ديجيتال مع التفسير  
- مبني باستخدام HTML, CSS, JavaScript  
- يحتوي على ملف tafsir.json لعرض التفسير  
- الهدف: تسهيل قراءة القرآن مع التفسير من أي جهاز  

## طريقة التشغيل
1. افتح المشروع في GitHub Codespaces أو أي محرر.  
2. شغل الملف index.html في المتصفح.  

## المميزات
- واجهة بسيطة وسهلة الاستخدام  
- دعم اللغة العربية بالكامل  
- إمكانية إضافة مزايا مستقبلية مثل البحث أو الوضع الليلي
